================================================================================
HOW TO RUN
================================================================================

1. Install dependencies:
   npm install --prefix server
   npm install --prefix client

2. Start the app:
   ./start.sh

3. Open http://localhost:5189

4. View logs:
   API: http://localhost:3021/health
   Client: Vite dev output in terminal

5. Try features:
   - Browse tab: Search and filter plants
   - Zone tab: Use your location or enter zone
   - New List tab: Create a list and add plants
   - Lists tab: View and expand your lists
   - Export tab: Generate PDF (killer feature!)
