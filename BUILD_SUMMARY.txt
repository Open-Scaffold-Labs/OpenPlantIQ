QUICK START
================================================================================

1. Install dependencies:
   npm install --prefix server
   npm install --prefix client

2. Start the app:
   ./start.sh

3. Open http://localhost:5189

4. Try the killer feature:
   - Create a list in "New List" tab
   - Add some plants
   - Go to "Lists" tab and view the list
   - Click "Export PDF"
   - Click "Generate PDF" to see professional output
